# Chapter 2 Quiz - Introduction to C++
1. Every complete statement ends with a<br>
    A) period<br>
    B) # symbol<br>
    C) semicolon<br>

    B) function main<br>

    ```
    72
    'A'
    "Hello World"

    ```
   are all examples of<br>
